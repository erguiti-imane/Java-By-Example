# Java Data Types 

Java has eight built-in data types, referred to as the Java primitive types.

| Type    |  Size   |        Default value |
|---------|:-------:|---------------------:|
| byte    | 1 byte  |                    0 |
| short   | 2 bytes |                    0 |
| int     | 4 bytes |                    0 |
| long    | 8 bytes |                   0L |
| float   | 4 bytes |                 0.0f |
| double  | 8 bytes |                 0.0d |
| char    | 2 bytes |             '\u0000' |
| boolean |  1 bit  |                 null |
